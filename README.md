# Telegram-бот для заказов

Бот получает уведомления о заказах с сайта и может работать как отдельный процесс. В Telegram у него есть меню с командами `/start` и `/status`, а после запуска чата показываются кнопки `/start` и «Статус».

## Настройка

1. Создайте бота через `@BotFather` и скопируйте токен.
2. Добавьте токен в файл `C:\Users\admin\Desktop\zhanklod\.env.local`:

```env
TELEGRAM_BOT_TOKEN=токен_от_BotFather
TELEGRAM_CHAT_ID=
TELEGRAM_SILENT=false
TELEGRAM_PROXY_URL=http://127.0.0.1:10808
```

3. Если этот токен раньше использовался через webhook, удалите webhook:

```powershell
npm run delete-webhook
```

4. Для локального запуска из этой папки используйте:

```powershell
npm run start:local
```

Чтобы только зарегистрировать команды в меню Telegram без постоянного запуска бота, выполните `npm run setup`.

Либо без npm:

```powershell
node --env-file="C:\Users\admin\Desktop\zhanklod\.env.local" bot.mjs
```

Процесс должен оставаться запущенным. После запуска откройте бота в Telegram и нажмите **Start**. Команды также доступны через кнопку **Меню** возле поля ввода.

Команды запуска используют локальный прокси Windows `127.0.0.1:10808`, потому что прямое соединение с Telegram на этом компьютере недоступно. Перед запуском убедитесь, что приложение VPN/прокси включено.

## Запуск на сервере и приём заказов от Layero

На сервере запускайте бота командой `npm start`. Прокси компьютера при этом не используется. Хостинг должен передать приложению переменную `PORT` и открыть публичный HTTPS-адрес.

Добавьте на сервере бота переменные:

```env
TELEGRAM_BOT_TOKEN=токен_от_BotFather
TELEGRAM_CHAT_ID=id_чата_или_группы
TELEGRAM_RELAY_SECRET=длинный_случайный_ключ
```

После публикации проверьте адрес:

```text
https://2-26-111-48.sslip.io/health
```

В Layero добавьте:

```env
TELEGRAM_RELAY_URL=https://2-26-111-48.sslip.io/telegram/order
TELEGRAM_RELAY_SECRET=тот_же_длинный_случайный_ключ
```

После этого заново опубликуйте сайт. Layero будет отправлять заказ на сервер бота, а сервер бота — в Telegram.

Сайт также читает `TELEGRAM_PROXY_URL` при локальной отправке заказов. В настройках Layero эту переменную добавлять не нужно.

## Получение ID чата

Откройте чат с ботом, отправьте сообщение и выполните:

```powershell
node --env-file="C:\Users\admin\Desktop\zhanklod\.env.local" get-chat-id.mjs
```

Скопируйте показанный `id` в `TELEGRAM_CHAT_ID`. Для отправки заявок в группу добавьте бота в группу, напишите там сообщение и повторите команду.

`TELEGRAM_SILENT=true` отключает звук уведомлений о новых заказах.
